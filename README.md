# TerminalHeroquest
Terminal rol game for linux at first

Heroquest
Juego de rol por consola que emula al heroquest. Permitirá crear retos propios y tendrá un mapa interactivo por el cual el heroe se moverá. Inicialmente está creado para un solo jugador contra la máquina que hará las veces de Malvado Brujo.
